Each integration has a healthcheck endpoint exposed via GET /api/geo/health. The dependencyStatus module aggregates health from all services and marks the overall system as degraded if any non-optional service is down. PostGIS is the only critical dependency — without it, the system cannot answer spatial queries. The Python bridge, FAISS, and OSM bridge are optional; the system degrades gracefully (skips semantic encoding/search) if they are unavailable.
