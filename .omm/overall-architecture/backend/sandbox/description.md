SQLSandbox validates and executes spatial SQL against PostGIS. Uses a postgisCatalog allowlist (src/skills/postgis/sqlSecurity.ts) to restrict SQL to safe SELECT patterns. Templates (areaCategoryHistogram, areaRingDistribution, areaRepresentativeSample, areaCompetitionDensity, areaH3Hotspots, areaAoiContext, areaLanduseContext) are loaded from .sql files and interpolated with WKT geometry tokens. Limits row results to configurable maxRows (default 200).
