Chat runtime components. SSEWriter streams typed SSE events (trace, job, stage, thinking, reasoning, intent_preview, pois, stats, refined_result, done, boundary, spatial_clusters, vernacular_regions, fuzzy_regions) to the frontend. DeterministicRouter performs rule-based intent classification from the user's query text and spatial context, resolving anchor sources (user_location, map_view, comparison_pair) and query types (nearby_poi, nearest_station, area_overview, similar_regions, compare_places).
