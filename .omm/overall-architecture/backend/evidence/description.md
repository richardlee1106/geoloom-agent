Evidence system transforms raw tool call results into structured views. EvidenceViewFactory creates typed views: POIListView, AreaOverviewView, ComparisonView, TransportView, BucketView, SemanticCandidateView. Renderer then turns a view into a human-readable answer string. Area insight views enrich answers with category histograms, ring distributions, H3 hotspot grids, AOI context, and land-use data.
