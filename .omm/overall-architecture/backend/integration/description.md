Integration adapters for external services. PostgisPool wraps the pg driver with query timeout and healthcheck. PythonBridge calls the spatial encoding microservice (RemoteFirstPythonBridge retries remote, falls back to local). FaissIndex calls the semantic vector service (RemoteFirstFaissIndex). OSMBridge calls the routing service. dependencyStatus tracks ready/degraded state for all external dependencies and surfaces it in health endpoints.
