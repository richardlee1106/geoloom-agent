GeoLoomAgent is the central orchestration engine. It receives a ChatRequestV4, routes the intent via DeterministicRouter, builds a spatial constraint from the request, runs a function-calling loop against registered skills, assembles evidence views from tool call results, and synthesizes an answer via the LLM provider. Key sub-components: AlivePromptBuilder (dynamic system prompt), ConfidenceGate (decides whether to use LLM answer or fallback), ConversationMemory (turn summarization), SessionManager (session lifecycle), and RuntimeMetrics (latency, SQL validation rates).
