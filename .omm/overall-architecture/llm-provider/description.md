External LLM API provider (Anthropic or OpenAI-compatible) used by the GeoLoomAgent to synthesize grounded answers and power the function-calling loop. Provider is selected via environment variable (ANTHROPIC_API_KEY, OPENAI_API_KEY, or MODEL_NAME for OpenAI-compatible endpoints). If no API key is configured, InMemoryLLMProvider acts as a mock that still executes skills deterministically but skips LLM synthesis.
