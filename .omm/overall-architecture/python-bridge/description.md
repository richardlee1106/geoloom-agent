Python microservice (RemoteFirstPythonBridge) that provides spatial encoding. Endpoints: encode_region_snapshot (generates feature tags from a region snapshot), encode_poi_profile (generates role tags for a representative POI), score_similarity (compares two region snapshots), encode_query (semantic encoding of the user's natural language query). The backend calls this service via HTTP; if unavailable it degrades gracefully and skips semantic encoding steps.
